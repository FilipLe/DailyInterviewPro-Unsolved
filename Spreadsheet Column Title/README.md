<b>Output:</b> AY<br>
<br>
<b>Input:</b> 52<br>
<b>Output:</b> AZ<br>
<br>
<b>Input:</b> 676<br>
<b>Output:</b> YZ<br>
<br>
<b>Input:</b> 702<br>

<br>
